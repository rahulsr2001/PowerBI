# Blinkit Sales Dashboard

This project visualizes key sales and operational metrics for Blinkit (India's Last Minute App), enabling data-driven decision-making for business optimization. The dashboard, built using Power BI, offers a comprehensive view of sales, outlet performance, and customer preferences.

Key Use Cases:

Management Insights: Identify high-performing outlet types and locations.

Customer Preferences: Understand the demand for Low Fat vs. Regular items.
